The Gemini API can generate text output when provided text, images, video, and audio as input. This guide shows you how to generate text using the generateContent and streamGenerateContent methods. To learn about working with Gemini's vision and audio capabilities, refer to the Vision and Audio guides. Now that you have explored the basics of the Gemini API, you might want to try: Vision understanding: Learn how to use Gemini's native vision understanding to process images and videos. Audio understanding: Learn how to use Gemini's native audio understanding to process audio files.
